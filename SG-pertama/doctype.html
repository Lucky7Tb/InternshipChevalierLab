Doctype HTML

!DOCTYPE html bukanlah sebuah tag di HTML

melainkan memberitahu ke browser bahwa file ini tipenya file apa

semua file html harus ada !DOCTYPE html

Walaupun tidak menggunakan doctype pun html pun tetap terbaca oleh browser

Tetapi dengan !DOCTYPE html kita menentukan HTML apa yang akan kita gunakan

Secara default kita akan menggunakan HTML5

Ada beberapa tag pada HTML5 yang sudah tidak di support. Seperti big dan center
